Try-Pods was my final project for the Lighthouse Labs Web Development Bootcamp. I teamed up with Rowan McKenzie and Kate Booton to build an app where users can share and discover podcasts. It's built with React and Ruby on Rails.

My profit and margin calculator is a simple JavaScript command line app that I use for my Shopify and Etsy businesses. It's incredibly customized and wouldn't be useful to anyone else, but for my purposes it works very well! It eliminates my need for spreadsheets and allows me to quickly calculate profit and margin when inventory costs, shipping costs, merchant fees, or prices change.

Tweeter is a simple, single-page Twitter clone. It was built during Lighthouse Labs' Web Development Bootcamp. It uses HTML, CSS, JS, jQuery, AJAX, Node, Express, Chance, and Body Parser, and features responsive design elements.
